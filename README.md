<h1>TURTLE (프로젝트 소개)</h1>
<h3>
거북이처럼 천천히, 꾸준히, 목표를 향해 조금씩 앞으로 나아가고 안전하게 목적지에 도달하는 것과 같이,<br>
확실한 협업을 통해 일의 각 단계를 신중하게 진행하고 조직원들의 아이디어와 능력을 조화롭게 결합하여<br>
지속적인 성장과 더 큰 성공을 이루어 낼 수 있도록 돕는 개발자용 협업 관리 도구 입니다.
